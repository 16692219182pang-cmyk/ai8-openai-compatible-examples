# Pricing and limits checklist

AI8 is promoted as a low-cost multi-model AI API gateway and offers signup credit. Do not rely on marketing copy alone. Verify these before long-term use:

- current model list
- model-specific pricing
- whether prices include relay/platform markup
- free/signup credit expiration
- rate limits
- refund policy
- uptime/SLA, if any
- data retention and privacy policy
- whether API keys can be scoped or rotated

For production traffic, start with a small workload and monitor cost, latency, and error rate.
