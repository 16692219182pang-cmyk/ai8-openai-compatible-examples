# Open WebUI setup

Open WebUI supports OpenAI-compatible endpoints.

Use these values:

```text
OpenAI API Base URL: https://ai8.my/v1
OpenAI API Key: your_ai8_api_key
```

If your AI8 dashboard shows a different API base URL, use that instead.

After saving, open the model list and run a small test prompt before using it for real workloads.
